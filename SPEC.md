# ペタコ 機能仕様書

## 1. 定型文管理

### 1.1 定型文一覧
- ウィンドウ左側に登録済み定型文を一覧表示する
- 各行にショートカットバッジ・テキスト・表示切替ボタン・削除ボタンを表示する
- 初回起動時はサンプル定型文を1件自動挿入する

### 1.2 追加・編集・削除
- 「追加」ボタンで新規定型文を作成する
- テキスト部分をクリックすると編集モーダルが開く
- 削除ボタン（×）で削除する

### 1.3 内容の非表示（マスク）
- 定型文ごとに非表示フラグ（`isHidden`）を持つ
- 非表示にすると一覧で内容の先頭4文字 + ●●●表示になる
- 目のアイコンをクリックすると一時的に内容を表示/再隠蔽できる

### 1.4 ショートカットキー
- 定型文ごとに独立したグローバルホットキーを設定する
- デフォルトは ⌘ + ⇧ + F1, F2, ... を空きキーから自動割り当て
- 修飾キーは ⌘ / ⇧ / ⌥ / ⌃ の組み合わせを許可する
- 一覧のショートカットバッジをクリックするとそのキーのみ変更するダイアログが開く
- ダイアログ中はすべてのホットキーを一時停止し、キーを取りこぼさない
- 同じキーが複数の定型文に割り当てられている場合、一覧上部に警告を表示する（最後に登録されたものが有効）

### 1.5 永続化
- `~/Library/Application Support/Petaco/snippets.json` に JSON で保存する
- v1 で数字キー割り当てだったデータは起動時にファンクションキーへ自動マイグレーションする

---

## 2. コピー履歴

### 2.1 記録
- クリップボードの変更を 0.4 秒ごとにポーリングして文字列を自動記録する
- 同じ内容が既にある場合は古いものを削除して先頭に移動する（重複排除）
- 最大 50 件を保持し、超えた分は末尾から削除する
- ペタコ自身が貼り付けのために書き込んだクリップボード変更は記録しない

### 2.2 表示・操作
- ウィンドウ右側に新しい順で一覧表示する（参照用。貼り付けはクイック選択オーバーレイから行う）
- 各行の × ボタンで個別削除する
- 「すべて削除」ボタンで確認ダイアログを経て全件削除する

### 2.3 永続化
- `~/Library/Application Support/Petaco/history.json` に JSON で保存する

---

## 3. クイック選択（オーバーレイ）

### 3.1 起動
- 登録したショートカットキーを押すとコピー履歴のオーバーレイを画面中央に表示する
- 履歴が空の場合はオーバーレイを表示しない
- 同じショートカットを再度押すとオーバーレイを閉じる

### 3.2 ショートカット登録
- クイック選択ショートカットは複数登録できる
- ウィンドウ右上の「追加」から追加・変更・削除する
- デフォルトは ⌘ + ⇧ + Space
- `UserDefaults` に JSON で永続化する

### 3.3 操作
| 操作 | 動作 |
|------|------|
| ↑ / ↓ | 選択移動（長押しでリピート: 0.4 秒後に 0.12 秒間隔） |
| Enter またはクリック | 選択中の内容を貼り付けてオーバーレイを閉じる |
| Esc | キャンセルして閉じる |
| その他のキー | キャンセルして閉じる |
| オーバーレイ外クリック | キャンセルして閉じる |

### 3.4 貼り付け先
- オーバーレイを開いた時点で最前面にいたアプリを貼り付け先として記憶する
- オーバーレイはフォーカスを奪わない（`nonactivatingPanel`）
- ↑↓/Enter/Esc は Carbon ホットキーで消費し、背後のアプリに渡さない

---

## 4. コピー通知

- テキストをコピーすると「コピーしました」通知バナーを表示する
- 本文は最大 50 文字でプレビュー表示する（超えた場合は「…」で省略）
- 同じ内容を連続でコピーした場合は通知を出さない
- 複数の pending 通知は ID `petaco.copy` で管理し、最新のもので上書きする
- 初回起動時に通知許可が必要（システム設定 > 通知 > ペタコ）

---

## 5. 貼り付け方式

- テキストを一時的にクリップボードへ書き込み、CGEvent で Cmd+V を送信する
- 貼り付け後、元のクリップボード内容を自動復元する
- ペタコ自身の書き込みは `ClipboardMonitor.ignore(changeCount:)` で除外し、コピー履歴に含めない

---

## 6. メニューバー常駐

- Dock には表示しない
- メニューバーのクリップボードアイコンで常駐する
- 左クリック → ウィンドウを前面に表示する
- 右クリック → 「ペタコを開く」「終了」メニューを表示する
- ウィンドウの赤い閉じるボタンを押すとアプリも終了する

---

## 7. ログイン時自動起動

- ウィンドウ上部の「自動的に立ち上げる」チェックボックスで設定する
- macOS 13 以降の `SMAppService` を使用する

---

## 8. 必要な権限

| 権限 | 用途 |
|------|------|
| アクセシビリティ | CGEvent で Cmd+V を送信するために必要 |
| 通知 | コピー通知バナーの表示に必要 |

---

## 9. 動作環境・技術スタック

| 項目 | 内容 |
|------|------|
| 対応 OS | macOS 13.0 以降 |
| 言語 | Swift / SwiftUI |
| ホットキー | Carbon `RegisterEventHotKey` |
| クリップボード監視 | `NSPasteboard.changeCount` ポーリング（0.4 秒間隔） |
| 通知 | `UNUserNotificationCenter` |
| 自動起動 | `SMAppService` |
| 永続化 | JSON ファイル（定型文・履歴）、`UserDefaults`（クイック選択ショートカット） |
