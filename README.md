# ペタコ

登録した定型文をショートカットキーでどこでも即貼り付けできる、macOS用メニューバーアプリです。

## 機能

- **定型文ショートカット** — 定型文ごとに独立したグローバルホットキー（デフォルト: ⌘⇧F1, ⌘⇧F2...）を割り当て、どのアプリにいても即貼り付け
- **クイック選択** — ⌘⇧Space でコピー履歴の一覧をオーバーレイ表示。↑↓ で選んで Enter または クリックで貼り付け
- **クリップボード履歴** — コピーした内容を自動記録（最大50件）。同じ内容は重複なく先頭に移動
- **貼り付けプレビュー** — 貼り付けボタンを長押しすると内容を確認してから実行できる
- **メニューバー常駐** — Dock には表示せず、メニューバーのアイコンで常駐。左クリックでウィンドウ表示、右クリックで終了
- **ログイン時自動起動** — macOS 13以降の SMAppService を使用

## 動作環境

- macOS 13.0 以降
- Xcode 15.3 以降（ビルド用）

## ビルド方法

1. `Petaco.xcodeproj` を Xcode で開く
2. ▶️ ボタンでビルド・実行

## 初回セットアップ

ショートカットキーによる貼り付けには **アクセシビリティ権限** が必要です。

1. アプリを初回起動する
2. ショートカットキーを押すと権限ダイアログが表示される
3. システム設定 > プライバシーとセキュリティ > アクセシビリティ で **ペタコ** を有効にする
4. アプリを再起動する

## 使い方

### 定型文の登録
「追加」ボタンで定型文を登録し、「キーを変更」で好きなキーを割り当てます。  
修飾キー（⌘/⇧/⌥/⌃）との組み合わせが必須です。

### クイック選択（コピー履歴）
| 操作 | 動作 |
|------|------|
| ⌘⇧Space | 履歴オーバーレイを開閉 |
| ↑ / ↓ | 選択移動（長押しでリピート） |
| Enter またはクリック | 貼り付けて閉じる |
| Esc・他のキー・オーバーレイ外クリック | キャンセルして閉じる |

クイック選択のショートカットはウィンドウ右上の「変更」ボタンで変更できます。

## 仕組み

- **貼り付け** — テキストを一時的にクリップボードへ書き込み、CGEvent で Cmd+V を送信。ペースト後は元の内容に自動で復元
- **ホットキー** — Carbon の `RegisterEventHotKey` を使ってアプリ共通のグローバルホットキーを登録
- **クイック選択のキー操作** — オーバーレイ表示中のみ有効な別セッションのホットキーを登録し、↑↓/Enter/Esc を背後のアプリに渡さず消費
- **クリップボード監視** — `NSPasteboard.changeCount` を 0.4 秒ごとにポーリング。ペタコ自身の書き込みは除外
- **永続化** — 定型文・履歴ともに `~/Library/Application Support/Petaco/` にJSONで保存

## 既知の制約

- 同じショートカットキーを複数の定型文に割り当てると警告が出ます（最後に登録されたものが有効）
- クイック選択のショートカットと定型文のショートカットが重複した場合の自動チェックはありません
