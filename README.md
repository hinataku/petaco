# ペタコ セットアップ手順

## このフォルダについて
`Petaco.xcodeproj` を含む、**そのままXcodeで開いてビルドできる状態**の
プロジェクト一式です。新規プロジェクト作成やファイルのドラッグ&ドロップは
不要です。

## 0. 既存の petaco フォルダがある場合
以前作成したXcodeプロジェクト（壊れている場合や、重複ファイルがある場合）を
お使いの場合は、Xcodeでそのプロジェクトを開いていれば一度閉じてから、
フォルダの中身を全て削除し、このzipを展開した `PetacoProject` フォルダの中身
（`Petaco.xcodeproj` と `Petaco` フォルダ）をそこに置き直してください。

## 1. Xcodeで開く
`Petaco.xcodeproj` をダブルクリックして Xcode で開いてください。
Carbon.framework は既にプロジェクトに追加済みです。

## 2. アクセシビリティ権限の設定（重要）
このアプリは他のアプリにテキストを貼り付けるため、初回実行時に
システム設定で許可が必要です。「クイック選択（⌘⇧Space）」機能のグローバルキー監視も
同じ権限を使用します。

1. アプリを一度ビルド・実行する
2. ショートカットキーを押すと、許可ダイアログが出る
3. システム設定 > プライバシーとセキュリティ > アクセシビリティ で
   `ペタコ` を有効にする
4. アプリを再起動する

## 3. ビルド & 実行
Xcode上部の ▶️ ボタンでビルド・実行。
ウィンドウが開いたら「追加」ボタンで定型文を登録し、
表示されているショートカットキー（デフォルトは ⌘⇧F1, ⌘⇧F2...）を
他のアプリ（メモ帳など）にフォーカスした状態で押すと、貼り付けが実行されます。

## クイック選択（⌘⇧Space）
ウィンドウを開かなくても、どこでも以下の操作で一覧から選んで貼り付けられます。

1. `⌘⇧Space` を押す → 画面中央に定型文＋履歴の一覧がオーバーレイ表示される
2. `↑` `↓` で選択を移動し、クリックまたは `Enter` で貼り付ける
3. もう一度 `⌘⇧Space` を押す、`Esc`・ほかのキーを押す、またはオーバーレイ外をクリックする → 閉じる

**注意:** `⌘⇧Space` は定型文個別のショートカットとして重複登録しないでください
（編集画面のキー設定で `⌘⇧Space` を選ばないようにしてください。今のところ自動チェックは
個別定型文同士の重複のみで、この機能専用キーとの衝突チェックは入っていません）。

このキー監視は通常のグローバルホットキー登録（Carbon API）とは別に、
`NSEvent` のグローバル/ローカルモニターを使って実装しています
（macOS標準の⌘V自体には一切介入しません。⌘⇧Spaceという別のキーのみを監視します）。

## 貼り付け前のプレビュー
各貼り付けボタン（クリップアイコン）は以下の動作をします。
- **クリック** : 即時貼り付け（今までと同じ）
- **長押し（0.5秒以上）** : 内容をポップアップでプレビュー表示。
  「貼り付け」を押すと実行、「キャンセル」を押すと何もせず閉じる

## メニューバー常駐
ペタコはDockには表示されず、メニューバーにクリップボードアイコンとして常駐します。
- 赤い閉じるボタンでアプリを終了できます
- メニューバーのアイコンをクリックすると、登録一覧ウィンドウが再表示されます
- 完全に終了したい場合は、メニューバーアイコンを**右クリック**して「終了」を選んでください

## ログイン時の自動起動
画面上部の「PC起動時に自動的に立ち上げる」チェックボックスをONにすると、
Mac起動・ログイン時にペタコが自動で立ち上がります。
（追加のHelperアプリ不要、macOS 13以降の `SMAppService` を使用）

初回ONにした際、システム設定側で許可が必要な場合があります。
その場合はシステム設定 > 一般 > ログイン項目 で `Petaco` が有効になっているか確認してください。

## 画面構成
- 左側: 登録済み定型文一覧（追加・編集・削除・クリップボードアイコンで貼り付け実行）
- 右側: 貼り付け履歴（ペタコ経由で貼り付けた内容が新しい順に並ぶ）
  - 同じ内容を再度貼り付けると、古い履歴は自動的に消えて最新の1件だけが残ります
  - 履歴からも直接「貼り付け」や個別削除、「すべて削除」が可能です

## 仕組みの概要
- `HotkeyManager` : Carbon APIで全アプリ共通のグローバルホットキーを監視
- `PasteManager`   : クリップボードに一時的にテキストをセットし、⌘Vを仮想的に送信してペースト。
  ペースト後は元のクリップボード内容に自動で戻します
- `SnippetStore`   : 定型文を `~/Library/Application Support/Petaco/snippets.json` に保存
- `PasteHistoryStore` : 貼り付け履歴を `~/Library/Application Support/Petaco/history.json` に保存
  （同一内容は古い方を削除し、最新のものだけ残します。最大50件まで保持）
- ショートカットキーは編集画面の「キーを変更」ボタンを押し、好きなキーを押すと
  そのキーに変更されます（修飾キー必須。⌘/⇧/⌥/⌃ のいずれかと組み合わせてください）

## 既知の制約
- 同じショートカットキーを複数の定型文に割り当てると、一覧に警告が出ます
  （最後に登録されたものだけが有効になります）
- アプリを完全終了するとホットキーも無効になります（バックグラウンド常駐させたい場合は
  Dockアイコンを消してメニューバー常駐型に変更することも可能です。必要なら相談してください）
