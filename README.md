# ペタコ

登録した定型文をショートカットキーでどこでも即貼り付けできる、macOS用メニューバーアプリです。

## インストール

1. [Petaco-1.0.0.zip](https://github.com/hinataku/petaco/blob/master/Petaco-1.0.0.zip) をダウンロード
2. zip を展開して `Petaco.app` を `/Applications` へ移動
3. 初回起動時に「開発元を確認できません」と表示される場合は、**システム設定 > プライバシーとセキュリティ** を開き「このまま開く」をクリック

## 機能

### 定型文ショートカット
- 定型文ごとに独立したグローバルホットキー（デフォルト: ⌘ + ⇧ + F1, F2...）を割り当て
- ウィンドウの定型文一覧から**ショートカットキーバッジをクリック**すると、そのキーだけを変更するダイアログが開く（ダイアログを開いただけでは変更されず、「入力し直す」を押してから新しいキーをキャプチャする）
- 同じキーが複数の定型文に割り当てられている場合は警告を表示

### コピー履歴とクイック選択
- コピーした文字列を自動記録（最大50件・重複排除）
- クイック選択ショートカット（デフォルト: ⌘ + ⇧ + Space）で画面中央に履歴の一覧をオーバーレイ表示
- ↑↓で選択し、クリックまたは Enter で貼り付け。Esc・他のキー・オーバーレイ外クリックで閉じる
- クイック選択のショートカットは**複数登録可能**。ウィンドウ右上から追加・変更・削除できる

### コピー通知
- テキストをコピーすると通知バナーを表示（「コピーしました」＋内容のプレビュー）
- 同じ内容を連続コピーしても通知は出ない（重複抑制）
- 初回起動時に通知の許可が必要（システム設定 > 通知 > ペタコ）

### 貼り付けプレビュー
- 一覧・履歴の各クリップアイコンを**長押し（0.5秒）**すると内容をポップオーバーで確認
- 「貼り付け」で実行、「キャンセル」で閉じる

### メニューバー常駐
- Dock には表示せず、メニューバーのクリップボードアイコンで常駐
- 左クリック → ウィンドウを前面に表示
- 右クリック → 「ペタコを開く」「終了」メニュー
- ウィンドウの赤い閉じるボタンでアプリも終了

### ログイン時自動起動
- ウィンドウ上部の「PC起動時に自動的に立ち上げる」で設定（macOS 13以降の SMAppService を使用）

## 動作環境

- macOS 13.0 以降
- Xcode 15.3 以降（ビルド用）

## ビルド方法

1. `Petaco.xcodeproj` を Xcode で開く
2. ▶️ でビルド・実行

## 初回セットアップ

ショートカット貼り付けには **アクセシビリティ権限** が必要です。

1. アプリを初回起動する
2. ショートカットキーを押すと権限ダイアログが表示される
3. システム設定 > プライバシーとセキュリティ > アクセシビリティ で **ペタコ** を有効にする
4. アプリを再起動する

## 使い方

| 操作 | 動作 |
|------|------|
| 定型文のショートカット（例: ⌘ + ⇧ + F1）| その定型文を最前面のアプリに貼り付け |
| クイック選択ショートカット（例: ⌘ + ⇧ + Space）| 履歴オーバーレイを開閉 |
| オーバーレイ中 ↑ / ↓ | 選択移動（長押しでリピート） |
| オーバーレイ中 Enter またはクリック | 選択項目を貼り付けて閉じる |
| オーバーレイ中 Esc・他のキー・外クリック | キャンセルして閉じる |
| クリップアイコン長押し（0.5秒） | 内容をプレビュー |

## 仕組み

- **貼り付け** — テキストを一時的にクリップボードへ書き込み、CGEvent で Cmd+V を送信。ペースト後は元の内容を自動復元。ペタコ自身の書き込みはコピー履歴に含めない
- **定型文ホットキー** — Carbon の `RegisterEventHotKey` でアプリ共通のグローバルホットキーを登録。ショートカット変更中は一時停止
- **クイック選択** — オーバーレイ表示中のみ有効な別セッションのホットキー（↑↓/Enter/Esc）を登録し、背後のアプリにキーを渡さず消費。貼り付け先アプリのプロセスへ直接 Cmd+V を送るためフォーカスが奪われない
- **クリップボード監視** — `NSPasteboard.changeCount` を 0.4 秒ごとにポーリング。ペタコ自身の書き込みは `ClipboardMonitor.ignore(changeCount:)` で除外
- **コピー通知** — `UNUserNotificationCenter` で通知バナーを送信。同じ内容の連続コピーは抑制。複数の pending 通知は最新のもので上書き
- **複数ショートカット** — `QuickPickShortcutStore` が `[QuickPickShortcut]` を管理し、`QuickPickManager` が全ショートカット分のホットキーを並列登録
- **永続化** — 定型文・履歴は `~/Library/Application Support/Petaco/` に JSON で保存。クイック選択ショートカットは `UserDefaults` に JSON で保存

## 既知の制約

- 同じショートカットキーを複数の定型文に割り当てると警告が出ます（最後に登録されたものが有効）
- 定型文ショートカットとクイック選択ショートカットの重複は自動チェックしていません
- アクセシビリティ権限がないと貼り付けが動作しません（ダイアログが表示されます）
